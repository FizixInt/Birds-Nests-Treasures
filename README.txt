Birds Nests & Treasures is a random treasure mod for Minecraft 1.14, inspired by the "Birds Nests" Mod by Clever Panda.

Birds Nests & Treasures adds a number of collectable treasures to the world that can be found while exploring; such as Birds Nests, Bandit Treasure Pouches, Sunken Treasure Bottles and Ancient Treasure. If you find any of these items, simply right click while holding to open them and contain the treasures within.
